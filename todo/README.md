# ToDo.

An interactive ToDo. App which enablees you to add tasks and get notified at that point of time.

## Existing Features

* Add a Task.
* Prioritize Tasks.
* Get Notified on the Task.
* Modify it.
* Delete it.

## Upcoming Features

* Archive Tasks.
* Soft Delete Tasks.
* Schedule tasks for duration of more than one day.
* Snooze Notifications.
* Categorize Tasks.
